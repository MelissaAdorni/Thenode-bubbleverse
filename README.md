# 🫧 Thenode-bubbleverse

Welcome to the **Thenode-bubbleverse**! This is a Node.js project designed to explore server-side logic in a bubbly way.

## 🚀 Getting Started
This project uses a simple HTTP server to say hello from the cloud.

### Prerequisites
- [Node.js](https://nodejs.org/) installed on your machine.

### How to Run
1. Clone this repository.
2. Run the following command in your terminal:
   ```bash
   node app.js
